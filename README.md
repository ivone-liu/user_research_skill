# ai-user-research Skill v5

v5 adds a **Persona State Update Mechanism**.

## What it changes

After persona interviews or other deeper analysis, the skill now requires:
- updating persona cards,
- versioning persona state,
- changing labels,
- revising segmentation,
- checking whether personas should merge or split,
- and refreshing overall conclusions when major fields change.

## Why this matters

Without state updates, persona work becomes stale almost immediately.
With state updates, the research becomes a living system:
the personas evolve as the understanding evolves.
