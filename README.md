# ai-user-research

Author: Ivone <ivone@nibbly.cn>  
License: MIT

`ai-user-research` is an evidence-grounded research skill for independent developers, product managers, and early-stage founders.

## Neutral example domains
- cycling
- running
- hiking
- strength training
- outdoor communities

## Information required before research

### Required
- product name
- one-line definition
- problem solved
- target users
- typical usage scenario
- current alternatives
- main validation question

### Strongly recommended
- geography
- pricing model
- product stage
- competitor list
- existing reviews / interviews / surveys
- whether a persona CSV exists
- what kind of judgment matters most this round

### Suggested prompt

```text
We are starting a new user research project.

Product name:
One-line definition:
Problem solved:
Target users:
Usage scenario:
Competitors / alternatives:
What I most want to validate:

Please:
1. structure the task
2. build a source plan
3. selectively use my persona CSV if relevant
4. generate 20+ personas
5. fill missing personas yourself if needed
6. output structured conclusions
```
