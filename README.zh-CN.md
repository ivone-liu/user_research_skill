# ai-user-research

作者：Ivone <ivone@nibbly.cn>  
协议：MIT

`ai-user-research` 是一套面向独立开发者、产品经理和早期创业者的证据约束型用户研究技能。

它可以帮助你：
- 明确研究任务
- 规划证据来源
- 生成 20+ 用户画像
- 结合公开样本与内置 / 用户上传的画像 CSV 基线库
- 在合适时只部分复用画像原型
- 在画像库不够时自动补齐新画像
- 在画像库不匹配时保留结构、不照搬内容
- 深入特定画像
- 更新画像状态
- 输出结构化结论

## 使用方式

1. 描述产品和最想验证的问题。
2. 如果你有评论、访谈、问卷或竞品讨论，可以一起提供。
3. 如果你有画像 CSV，也可以上传。
4. 让 skill 生成 20+ 画像。
5. 选择某些画像深访。
6. 更新画像状态。
7. 输出结构化结论。

```text

产品名称：

2. 设计证据来源计划
3. 有条件地参考我的画像 CSV

5. 如果不够就按相同结构补齐新画像

```

## CSV 会如何被使用

画像 CSV 会被视为**候选原型池**，不是市场需求证据。

技能会在三种模式中择一：

- 部分复用模式
- 结构复用模式
- 公开样本主导模式

也就是说：
- 有匹配的，只取匹配的一部分
- 不够的，由 AI 按同样结构补齐
- 不匹配的，不照搬内容，只借字段结构和组织方式
