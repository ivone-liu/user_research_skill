# ai-user-research

作者：Ivone <ivone@nibbly.cn>  
协议：MIT

`ai-user-research` 是一套面向独立开发者、产品经理和早期创业者的证据约束型用户研究技能。

它可以帮助你：
- 明确研究任务
- 规划证据来源
- 生成 20+ 用户画像
- 结合公开样本与可选画像 CSV 基线库
- 深入特定画像
- 更新画像状态
- 输出结构化结论

## 使用方式

1. 描述产品和最想验证的问题。
2. 如果你有评论、访谈、问卷或竞品讨论，可以一起提供。
3. 如果你有画像 CSV，也可以上传。
4. 让 skill 生成 20+ 画像。
5. 选择某些画像深访。
6. 更新画像状态。
7. 输出结构化结论。

### 推荐起手提示词

```text
我们开始一个新的用户研究项目。

产品名称：
一句话定义：
解决的问题：
目标用户：
使用场景：
竞品/替代方案：
我最想验证的问题：

请：
1. 梳理研究任务
2. 设计证据来源计划
3. 如果我的画像 CSV 有用就纳入参考
4. 生成 20+ 用户画像
5. 输出结构化结论
```

## CSV 会如何被使用

用户提供的画像 CSV 会被视为画像基线库，不是市场需求证据。

它能帮助：
- 提升画像覆盖度
- 提供相近原型召回
- 作为分群种子
- 提升深访原型可信度
